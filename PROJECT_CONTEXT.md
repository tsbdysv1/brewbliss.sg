# PROJECT_CONTEXT

## What this project is

`brewbliss.sg` is the new website codebase for BrewBliss Coffee in Ho Chi Minh City.

Current stack:
- Vite
- React
- React Router
- ESLint
- Vercel-ready deployment

## Product direction

This site should feel:
- warm
- modern
- coffee-first
- urban Saigon
- photo-friendly
- menu-forward
- easy to update later with more complete product and brand info
